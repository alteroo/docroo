Quick Start
========================================

Download docroo
------------------
::

    wget https://github.com/alteroo/docroo/archive/master.zip
    unzip master.zip
    cd docroo-master/


Run the installer
--------------------
::

    sh install.sh


Customize 
------------------------

Edit `conf.py` and `app.yaml`

Add your docs
----------------

Add your documentation as .rst files

Deploy
----------
To deploy, run the following command from the documentation directory
::

    make docsite
